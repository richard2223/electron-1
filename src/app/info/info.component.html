Welcome to the Micro-ui Starter Project



By using this project, you benefit from the following:

  • Inheriting Layout, Styles and practices from Celine
  • The ability to independently deploy Angular Modules
  • Built in Authentication and Role Management
  • Seamless integration of components from the Component Library
  • Multiple themes
  • Multiple Layouts

  • Upcoming changes, features of latest versions of Celine and Micro-ui starter projects
  • Documentation around the benefits of using micro-service architecture for the fontends
  • Documentation regarding how this app is structured, design decisions and how to use the component library
  • Steps regarding application deployment
  • Guidelines for contributing back to the component Library
